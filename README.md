# TSPro
TypeScript Pro - Basic Concept of Typescript to Understanding Design Pattern for TS projects

## Install Packages
Typescript -> for programming using TS and ts-node is for compiling typescript. Once you install ts-node you should be able to compile Ts files using command line.
> npm install -g typescript ts-node

Comand line to compile Ts file:
> tsc --help
> tsc index.ts
